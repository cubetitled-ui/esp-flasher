
# ESP Flasher 🔧

**Простая утилита для прошивки ESP32 — для новичков и профессионалов.**

## 📥 Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/cubetitled-ui/esp-flasher.git
   cd esp-flasher
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите:
   ```bash
   python main.py
   ```

## 🚀 Как пользоваться
1. **Вставьте ссылку** на GitHub (страницу релиза или прямую ссылку на `.bin`).
2. **Или выберите файл** — `.bin`, `.zip` или `.ino`.
3. **Подключите ESP32** по USB.
4. **Нажмите «⚡ Прошить!»**

Программа автоматически:
- ✅ Найдёт плату на COM-порту.
- ✅ Установит драйверы (CP210x, CH340).
- ✅ Скачает Arduino CLI и ядро ESP32 (если нужен `.ino`).
- ✅ Скомпилирует `.ino` в `.bin`.
- ✅ Прошьет плату через `esptool`.
- ✅ Перезагрузит плату.

## 📦 Поддерживаемые форматы
- `.bin` — готовая прошивка.
- `.zip` — архив с `.bin` внутри.
- `.ino` — скетч Arduino (компилируется автоматически).
- Ссылка на GitHub — автоматически ищет `.bin` в релизах.

## 🛠 Сборка в один .exe файл
1. Запустите `build.bat`.
2. Результат: `dist/ESP_Flasher.exe` — один файл, работает без установки.

## ❓ Решение проблем
- **Плата не найдена**: Проверьте подключение USB и драйверы (CP210x/CH340).
- **Ошибка компиляции**: Убедитесь, что установлен Arduino CLI и ядро ESP32.

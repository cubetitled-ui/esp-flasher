# ESP Flasher 🔧

Программа для прошивки ESP32 — для новичков.

## Как пользоваться

1. **Вставь ссылку** на GitHub (страницу релиза или прямую ссылку на `.bin`)
2. **Или выбери файл** — `.bin`, `.zip` или `.ino`
3. **Подключи ESP32** по USB
4. **Нажми «⚡ Прошить!»**

Всё остальное программа сделает сама:
- ✅ Найдёт плату на COM-порту
- ✅ Установит драйверы (CP210x, CH340)
- ✅ Скачает Arduino CLI и ядро ESP32 (если нужен `.ino`)
- ✅ Скомпилирует `.ino` в `.bin`
- ✅ Прошьет плату через esptool
- ✅ Перезагрузит плату

## Цвета статуса

| Цвет | Значение |
|------|----------|
| ⚪ Серый | Готов к работе |
| 🟡 Жёлтый | Ищу плату / Компилирую |
| 🔵 Синий | Прошиваю |
| 🟢 Зелёный | Готово! |
| 🔴 Красный | Ошибка |

## Сборка в один .exe файл

```
build.bat
```

Результат: `dist/ESP_Flasher.exe` — один файл, работает без установки.

## Поддерживаемые форматы

- `.bin` — готовая прошивка
- `.zip` — архив с `.bin` внутри
- `.ino` — скетч Arduino (компилируется автоматически)
- Ссылка на GitHub — автоматически ищет `.bin` в релизах
